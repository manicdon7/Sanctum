# Sanctum Audio Assets

This directory contains the ambient sound files for Sanctum:

- rain.mp3 - Rain sounds
- forest.mp3 - Forest ambiance  
- ocean.mp3 - Ocean waves
- fireplace.mp3 - Crackling fire
- night-crickets.mp3 - Night cricket sounds
- lofi1.mp3 - Lo-fi instrumental track 1
- lofi2.mp3 - Lo-fi instrumental track 2

All files should be high-quality, seamlessly looping audio files optimized for mobile playback.

## Audio Requirements
- Format: MP3 or M4A
- Quality: 128kbps minimum, 320kbps maximum
- Length: 2-10 minutes per track for smooth looping
- Volume: Normalized and mastered for consistent levels
- Fade: Smooth fade in/out for seamless transitions
